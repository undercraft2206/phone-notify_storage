# phone-notify_storage
you get a notification on your phone when you start your, you will see hes storage, the storage left, the storage used and percentage of storage used


dependances:


-ntfy on android with play store


on the main file, you need to add the ip to the sentence: [IP], the port to the sentence: [PORT], the username to the sentence: [USER]


you can compile it with gcc: 


bash ''
gcc ntfy_storage.c -o ntfy_storage
''

and execute it with


bash ''
./ntfy_storage
''
